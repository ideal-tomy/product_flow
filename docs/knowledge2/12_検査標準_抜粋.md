# 検査標準(QC-STD-02)抜粋 株式会社ミナトテック
第2層文書 / 最終改定: 2026年5月10日 / 管理部署: 品質保証部 / 運用: 品質管理課(内線610)

## 1. 適用範囲
厚木工場における受入検査・工程内検査・最終検査に適用する。判定基準の詳細は本標準を正とし、作業手順書の記載と異なる場合は品質管理規定 第4条に従う。

## 2. 受入検査
- 対象: 基板(PWB)、電子部品、筐体部品
- 抜取方式: JIS Z 9015 準拠、なみ検査水準II、**AQL 0.65**
- 不合格ロットは受入保留とし、品質管理課長が処置(返品/全数選別/特採申請)を判定する。
- 検査記録はMT-PROへ当日中に登録する。

## 3. 工程内検査(SMT実装)
### 3.1 リフロー後 目視・AOI検査
- AOI(自動光学検査)は全数実施。AOI指摘品は認定検査員が目視再判定する。
- はんだ接合の判定基準:
  - **フィレット高さ: リード厚の1/2以上**
  - 濡れ角: 90度以下
  - ボイド: 接合面積の25%以下(BGAは30%以下)
- ブリッジ・未はんだ・部品欠落は即不適合。手直しは認定作業者のみ可(1箇所につき手直しは2回まで)。

### 3.2 温度プロファイル確認
- リフロー炉の温度プロファイルは**各シフト開始時**および条件変更後に確認し、チェックシートをMT-PROへ登録する。
- 規格: ピーク温度 240〜250℃ / 220℃以上の保持時間 30〜60秒 / 昇温勾配 3℃/秒以下
- 規格外を検出した場合: 直前の確認以降に流動したロットを疑義品とし、赤棚へ隔離のうえ品質管理課へ連絡(品質管理規定 第7条)。

## 4. 工程内検査(組立)
- ねじ締結: トルクドライバー使用、規定トルクは各SOPに定める。トルクドライバーは始業時に校正確認済みシールの有効期限を確認する。
- 圧着端子: 引張強度試験を各ロット先頭5本で実施、最小値 80N 以上。

## 5. 最終検査
- 電気特性検査: 全数実施(MT-PRO試験プログラム)
- 外観検査: 抜取(AQL 1.0)。傷の限度は限度見本(検査台に掲示)による。
- 出荷判定: 品質管理課長の承認をもって出荷可とする。

## 6. 検査記録の保管
検査記録・チェックシートの保管期間は7年とする(顧客契約でこれを超える指定がある場合は契約に従う)。
