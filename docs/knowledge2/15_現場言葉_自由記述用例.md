# 現場言葉・自由記述用例（製造① minato-factory）

正本ルール: [`../field-language-interpretation-rules.md`](../field-language-interpretation-rules.md)  
ガイド台本: [`14_想定質問リスト_製造現場デモ台本.md`](./14_想定質問リスト_製造現場デモ台本.md)

ガイド外の「現場一言」を想定。サンプルは `questionId` へエイリアス、ライブは同条を根拠に回答。

---

### F1. 合格品の中に塗装剥がれあったがどうする？

- **intent:** 外観不適合（疑い）の発見 → 不適合対応
- **期待回答の骨子:** 手直し・廃棄せず中断 → 赤票 → 赤棚 → 内線610 → MT-QAポータル当日登録 → 処置は品質管理課長
- **必須参照:** QM-R-01 第7条（＋外観は QC-STD-02 §5 限度見本）
- **サンプル着地:** `nc-handling`
- **NG:** 「出荷してよい／自分で直してよい」と推測

### F2. 傷ついてる基板見つけた。捨てていい？

- **intent:** 廃棄の自己判断禁止
- **期待回答の骨子:** 廃棄は不可。隔離・連絡・報告後、課長判定
- **必須参照:** 第7条
- **サンプル着地:** `nc-handling`
- **NG:** 廃棄可否を文書外で断定

### F3. 自分でヤスリかけて直していい？／ちょっと手直ししたい

- **intent:** 自己手直し禁止（認定・判定前）
- **期待回答の骨子:** 自己判断の手直し禁止。隔離→QC。手直しは判定後・認定作業者・回数制限
- **必須参照:** 第7条 / QC-STD-02 §3.1 / SOP §6
- **サンプル着地:** `nc-handling`
- **NG:** 「軽微なら直してよい」

### F4. 出荷前にキズ見つけた。もう検査通ってるんだけど

- **intent:** 合格後発見でも不適合疑い扱い
- **期待回答の骨子:** タイミング問わず第7条。隔離しQCへ
- **必須参照:** 第7条
- **サンプル着地:** `nc-handling`
- **NG:** 「検査通ってるから出荷可」

### F5. 不良っぽいんだけどライン止めていいの？

- **intent:** 異常時の停止＋隔離
- **期待回答の骨子:** 停止・ライン長報告。疑い品は赤棚＋610。自己再投入禁止
- **必須参照:** SOP §6 / 第7条
- **サンプル着地:** `nc-handling`
- **NG:** 停止不要と推測

### F6. 品質に電話したいんだけど、どっちの番号？

- **intent:** QC vs QA
- **期待回答の骨子:** 日常判定・検査は610(QC)、4M・特採・文書・顧客は620(QA)
- **必須参照:** 工場設定・組織
- **サンプル着地:** `qc-vs-qa`
- **NG:** 窓口を一本化して誤案内

### F7. 温度ちょっと上げたいんだけど、いいよね？

- **intent:** 4M（設備条件）変更
- **期待回答の骨子:** 不可。MT-QA申請→QA承認。緊急は製造部長暫定＋24h事後申請
- **必須参照:** 第9条 / SOP §4
- **サンプル着地:** `conveyor-change`
- **NG:** 「ちょっとなら可」

### F8. 手順書と検査標準で数字違う。どっち？

- **intent:** 文書優先
- **期待回答の骨子:** 規定>標準>手順。標準に従い、QA(620)へ不整合報告
- **必須参照:** 第4条（保持時間例は QC 3.2 vs SOP 5.2）
- **サンプル着地:** `hold-time-conflict`
- **NG:** 現場都合で手順を優先

### F9. ノギスのシール、先月切れてた。昨日測っちゃったかも

- **intent:** 校正期限切れ使用
- **期待回答の骨子:** 使用禁止・黄タグ。使用判明→疑義品・直ちに610へ報告
- **必須参照:** 第11条
- **サンプル着地:** `calibration-expire`
- **NG:** 「誤差なければ問題なし」

### F10. 新人にリフローやらせていい？

- **intent:** 特別工程の認定
- **期待回答の骨子:** 不可。認定者のみ。未認定配置は4M(人)
- **必須参照:** 第13条 / 第9条 / SOP §2
- **サンプル着地:** `certified-operator`
- **NG:** 「見てれば可」

### F11. 規格外の部品、今回だけ使いたい

- **intent:** 特採
- **期待回答の骨子:** QA部長承認。顧客仕様なら顧客書面も。特採番号をMT-PROへ。同一事由3回まで
- **必須参照:** 第8条
- **サンプル着地:** ライブ優先（サンプル未登録時は第8条チャンクで回答）
- **NG:** 現場長の口頭OKで可とする

### F12. RF-2号機の修理費っていくらだった？（範囲外）

- **intent:** ナレッジ外（金額）
- **期待回答の骨子:** 回答不可。製造・設備担当へ確認案内
- **必須参照:** なし
- **サンプル着地:** 拒否
- **NG:** 金額の推測
