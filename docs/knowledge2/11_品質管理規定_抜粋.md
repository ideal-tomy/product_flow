# 品質管理規定(QM-R-01)抜粋 株式会社ミナトテック
第1層文書 / 最終改定: 2026年3月1日 / 管理部署: 品質保証部(内線620)

## 第4条(文書の優先順位)
品質関連文書は「規定 > 標準 > 作業手順書」の順に優先する。
文書間の矛盾・不整合を発見した者は、作業を上位文書に従って行うとともに、速やかに品質保証部(内線620)へ報告する。

## 第7条(不適合品の管理)
1. 不適合品(疑いを含む)を発見した作業者は、**その場で作業を中断し、自己判断で手直し・廃棄をしてはならない**。
2. 発見者は不適合品に赤色の識別票を貼付し、工程脇の「不適合品置場(赤棚)」へ隔離する。
3. 発見者またはライン長は品質管理課(内線610)へ連絡し、MT-QAポータルへ不適合報告を当日中に登録する。
4. 処置(手直し/特別採用/廃棄)の判定は品質管理課長が行う。

## 第8条(特別採用【特採】)
1. 規格を外れた製品・材料をやむを得ず使用する場合は、特別採用の承認を要する。
2. 承認権限: 品質保証部長。顧客仕様に関わる場合は**顧客の書面承認**も必要とする。
3. 特採品はロット記録(MT-PRO)に特採番号を紐付け、トレーサビリティを確保する。
4. 同一事由の特採は3回までとし、以降は恒久対策の完了を条件とする。

## 第9条(4M変更管理)
1. 4M(人・設備・材料・方法)の変更は、事前にMT-QAポータルから変更申請を行い、品質保証部の承認を得るまで実施してはならない。
2. 事前申請の対象例: 装置条件(温度・速度等)の変更 / 材料メーカー・ロットグレードの変更 / 作業手順の変更 / 未認定作業者の配置
3. 承認区分: 軽微変更は品質保証部担当者、重要変更(顧客要求仕様に影響)は品質保証部長+必要に応じ顧客通知。
4. 緊急時(設備故障等)は製造部長の判断で暫定変更できるが、24時間以内にMT-QAポータルへ事後申請する。

## 第11条(測定機器の管理)
1. 検査・測定に使用する機器は校正管理の対象とし、校正周期は原則12か月とする(機器台帳: MTドキュメント)。
2. 校正期限切れの機器は使用禁止とし、黄色の「使用禁止」タグを貼付する。
3. **校正期限切れ機器での測定が判明した場合**、当該機器で測定した期間のロットを疑義品として扱い、品質管理課が再検証する。判明した者は直ちに品質管理課(内線610)へ報告する。

## 第13条(教育・認定)
1. 特別工程(はんだ付け・圧着)の作業は、社内認定を受けた認定作業者のみが行う。
2. 認定の有効期間は2年とし、更新には実技試験の合格を要する。
3. 未認定者の配置は第9条の4M変更(人の変更)として扱う。

## 第15条(是正処置)
重大不適合(顧客流出・同一不適合の月3件以上再発)は、品質保証部主催の是正処置会議を開催し、発生から10営業日以内に是正処置報告書を提出する。
