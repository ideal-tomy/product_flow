# 就業規則(抜粋) 株式会社ミナトテック
最終改定: 2026年4月1日 / 管理部署: 総務部

## 第2章 勤務

### 第8条(所定労働時間)
所定労働時間は1日8時間、休憩60分とする。
- 本社・大阪営業所: 9:00〜18:00(休憩12:00〜13:00)
- 厚木工場: 8:30〜17:30(休憩12:00〜13:00)

### 第9条(フレックスタイム制)
本社勤務の正社員はフレックスタイム制を選択できる。
- コアタイム: 10:30〜15:00
- フレキシブルタイム: 7:00〜10:30 / 15:00〜21:00
- 適用申請はMTポータルから前月15日までに行い、総務部長の承認を要する。
- 厚木工場勤務者はライン勤務のため対象外。

### 第10条(時間外労働)
時間外労働は所属長の事前承認を要する。事前承認なく行った時間外労働は原則認めない。
月45時間を超える見込みが生じた場合、所属長は速やかに総務部(内線201)へ報告する。

### 第12条(在宅勤務)
正社員は週2日を上限として在宅勤務を行うことができる。
- 前週金曜17:00までにMTポータルで申請し、所属長の承認を得る。
- 入社後6か月未満の社員、および製造部は対象外。
- 在宅勤務手当として1日あたり250円を支給する(経費精算規程 第14条参照)。

## 第3章 休暇

### 第15条(年次有給休暇)
- 入社6か月経過後に10日を付与し、以後法令に従い勤続年数に応じて付与する。
- 取得は原則3営業日前までにMTポータルで申請する。
- 前日・当日の申請は所属長が業務に支障がないと認めた場合に限り認める。
- 時間単位有給は年5日分まで取得可能(1時間単位)。

### 第16条(慶弔休暇)
- 本人の結婚: 5日 / 子の結婚: 2日
- 配偶者・子・父母の死亡: 5日 / 祖父母・兄弟姉妹の死亡: 3日
- 申請はMTポータルから行い、総務部(矢島・内線203)へ連絡する。

### 第17条(その他の休暇)
- リフレッシュ休暇: 勤続10年・20年到達時に連続5日(福利厚生ガイド参照)
- 裁判員休暇: 必要日数(有給扱い)

## 第5章 服務

### 第25条(副業)
副業を行う場合は事前にMTポータルから届出を行い、総務部長の許可を得る。
競業に当たる業務、および週20時間を超える副業は許可しない。

### 第28条(退職)
自己都合退職の場合、退職日の1か月前までに所属長へ退職届を提出する。
