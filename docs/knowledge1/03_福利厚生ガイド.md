# 福利厚生ガイド 株式会社ミナトテック
2026年度版 / 管理部署: 総務部(内線202)

## 1. お祝い金・お見舞金
| 種類 | 金額 | 申請方法 |
|---|---|---|
| 結婚祝金 | 30,000円 | MTポータル+戸籍記載事項証明書等 |
| 出産祝金 | 第1子 30,000円 / 第2子以降 50,000円 | MTポータル |
| 傷病見舞金 | 連続14日以上の療養で 20,000円 | 総務部へ連絡 |
| 弔慰金 | 配偶者・子・父母 50,000円 / その他二親等 20,000円 | 総務部(矢島・内線203)へ連絡 |

※申請期限はいずれも事由発生から6か月以内。

## 2. リフレッシュ休暇
- 勤続10年: 連続5日の特別休暇 + 旅行補助 50,000円
- 勤続20年: 連続5日の特別休暇 + 旅行補助 100,000円
- 到達年度の翌年度末までに取得すること。旅行補助は旅行代金の領収書をMT経費から申請。

## 3. 資格取得支援
- 会社が指定する資格(対象リストはMTポータル掲載)に合格した場合、受験料全額+報奨金を支給。
- 報奨金例: 基本情報技術者 20,000円 / TOEIC 800点以上 30,000円 / 衛生管理者 15,000円
- 受験前にMTポータルから「資格取得支援申請」を行うこと。事前申請がない場合は支給対象外。

## 4. 健康支援
- 定期健康診断: 年1回(毎年10月、就業時間内・費用会社負担)
- 人間ドック補助: 35歳以上の希望者に上限 25,000円/年
- インフルエンザ予防接種: 全額会社負担(毎年11月に社内で集団接種を実施)

## 5. 住宅・生活
- 住宅手当: 世帯主かつ賃貸の場合 月15,000円(通勤圏内在住が条件)
- 引越し費用補助: 会社都合の転勤時は実費全額、自己都合で通勤時間が30分以上短縮される転居は上限 50,000円

## 6. 社内販売・その他
- 自社製品の社員割引: 定価の30%引き(年2回の社内販売会)
- 提携スポーツクラブ: 法人価格(月額 4,950円)で利用可。申込はMTポータル。

## 問い合わせ先
制度全般: 総務部(内線202) / お祝い金・弔慰金: 矢島(内線203) / 支給・精算: 経理部(内線301)
